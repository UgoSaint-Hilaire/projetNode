- Outils Nécéssaires pour le fonctionnement de l'application : 
	NodeJs 16, NPM 8.1.4, express 4.16

- Comment tester l'App:

	1- Configurer la base de données Mongo DB
		- Ajouter votre chaine de connexion de BD MongoDB dans le fichier .env (voir les slides du cours pour la création de Compte et de BD MongoDB)

	2 - Etant dans le dossier myapp:
		- Taper npm install dans un terminal, pour installer les dépendances
		- Ensuite, taper npm start pour démarer le server
		- Puis, http://localhost:3000/ dans un navigateur pour accéder à l'interface d'accueil du site
